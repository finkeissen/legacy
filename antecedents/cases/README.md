# Cases (Antecedents)

This directory contains **append-only case records**.

Each case is stored under a numeric folder (e.g. `000123/`) and includes:
- `gate.md`  (the minimal record)
- `links.md` (optional pointers to sources or related cases)

Cases may reference one or more failure modes defined in `antecedents/failure-modes/`.

No reconstruction, no fixes, no migration to matrix.
