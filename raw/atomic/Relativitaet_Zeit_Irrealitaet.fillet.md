# Relativität – Rahmenabhängigkeit ≠ Irrealität von Zeit

## Core Claim / Pattern
Relativitätstheorie zeigt, dass Zeitmessungen von Bezugssystemen abhängen (Zeitdilatation, Gleichzeitigkeit).

## Frequent Overinterpretation
Das wird zu ‚Zeit ist Illusion‘ oder ‚Zeit existiert nicht‘ übersteigert.

## Where the Interpretive Transfer Occurs
Von Koordinaten-/Rahmenabhängigkeit von Messgrößen wird auf Nicht-Existenz des Phänomens geschlossen.

## Philosophical Residue
Relativität liefert eine Strukturtheorie der Raumzeit; Ontologie (Eternalismus, Präsentismus etc.) ist zusätzliche Philosophie.

## Why the Misinterpretation Is Tempting
Weil ‚keine absolute Gleichzeitigkeit‘ wie ‚kein Jetzt‘ klingt.

## Useful Checks
- **Bridge principle?** Welche zusätzliche Annahme verbindet das formale/epistemische Ergebnis mit einer Aussage über die Wirklichkeit?
- **Scope?** Gilt der Satz über *Modelle/Begriffe/Methoden* oder über *Sein*?
- **Counterfactual:** Was müsste sich ändern, damit das Resultat nicht mehr gilt — die Welt oder das Rahmenwerk?
