# Information Theory — Fillet

## 1. Core Formal Structure
Information theory (Shannon) provides measures of uncertainty and communication limits:
- entropy,
- mutual information,
- channel capacity,
- coding theorems.

These are formal results about signals, codes, and probability distributions under specified assumptions.

## 2. Standard Interpretation
“Information” is frequently read as:
- meaning,
- knowledge,
- or an ontological substance.
Bits are treated as fundamental building blocks of reality (“it from bit”).

## 3. Illicit Upgrades
- **Entropy → meaning**: equating uncertainty reduction with semantic content without an explicit semantics layer.
- **Bits as units → bits as atoms**: moving from measurement convention to ontology.
- **Communication limits → cognitive limits**: treating channel bounds as limits on understanding in general.
- **Formal measure → explanatory essence**: using “information” as an explanation rather than as a quantified constraint.

## 4. Failure Modes
- Semantic drift: using the same word “information” for Shannon, Bayesian evidence, and meaning.
- Treating compression as explanation (“it’s simple, therefore true”).
- Importing agency into sources/channels (signals “want” to maximize information).

## 5. What Survives
- Shannon measures as **engineering constraints** and **diagnostic statistics**.
- Clear separation of: signal, code, distribution, and interpretation.
- Legitimate bridges only via explicit models (e.g., probabilistic semantics, data-generating processes).

## 6. How It Reconnects
- Connects as: **Constraint Layer** on communication/inference components in the Matrix.
- Requires explicit declaration of: what random variables represent, how symbols map to observations, and where semantics (if any) is introduced.
- Guardrail: no “it from bit” upgrade without extra, declared assumptions (and even then: methodological, not ontological).
