# Topics (Raw Material – Entry Points)

This document defines a **top-down topic map** for collecting unprocessed
historical and legacy material in `raw/`.

Topics describe **where reasoning typically originates**,
not whether it is correct, admissible, useful, or justified.

This map is intentionally **broad, non-evaluative, and incomplete**.

---

## Important Scope Disclaimer

This topic map:

- does **not** evaluate admissibility  
- does **not** trigger STOP or ALLOW  
- does **not** guide diagnostic runs  
- does **not** replace or pre-empt admissibility criteria  
- does **not** reflect the current public test phase scope  
  (e.g. number of domains or runs)

It exists solely to structure the **collection of raw material**.

Admissibility assessment, diagnostic runs, and STOP/ALLOW decisions occur
**exclusively outside `raw/`**.
