# Löwenheim–Skolem Theorem

## Core Result
The Löwenheim–Skolem theorem states that if a first-order theory has an infinite model, then it has a countable model. This is a model-theoretic result about the expressive limits of first-order logic.

## Frequent Overinterpretation
The theorem is often taken to imply that all infinite mathematical structures are “really” countable, or that set theory is paradoxical or inconsistent.

## Where the Interpretive Transfer Occurs
The mistake is to move from model-theoretic relativity to ontological relativism. The theorem shows that first-order axioms cannot control cardinality uniquely, not that cardinality itself is illusory or contradictory.

## Philosophical Residue
The theorem motivates stronger logics (e.g. second-order logic) or reflection on the limits of formal description, but it does not undermine the coherence of infinite mathematics.

## Why the Misinterpretation Is Tempting

The apparent paradox associated with the Löwenheim–Skolem theorem arises from a clash between two perspectives: the internal viewpoint of a model and the external viewpoint of model theory.

Within a model of set theory, notions such as “uncountable” are defined internally. From outside the model, however, we can describe the entire structure using a countable set of symbols. This external description can misleadingly be read as a claim about what exists inside the model itself.

The temptation is to collapse this distinction and treat limitations of first-order description as statements about mathematical reality. Once this collapse occurs, model-theoretic relativity is mistaken for ontological relativism.

Recognizing the separation between internal structure and external description dissolves the paradox. The theorem does not show that uncountable sets are illusory; it shows that first-order axioms cannot uniquely fix cardinality across all models. The intuitive unease reflects a shift in perspective, not a contradiction in mathematics.
