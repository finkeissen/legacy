# Structural (Legacy)

This directory collects historical material at the **structural level**
that preceded the current admissibility framework.

Purpose:
- Document failure modes, hidden assumptions, or scope violations
- Preserve historical context without rehabilitating positions
- Explain *why* certain approaches could not be carried forward

Rules:
- No reconstruction or optimization of legacy ideas
- No bridges into the current framework
- No normative or justificatory claims

This material is frozen history, not deferred work.
